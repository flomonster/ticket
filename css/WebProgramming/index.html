Sejong University Restaurants

Project

This project consist on developping a website which will reference all the restaurants in Sejong University.
This website will regroup the menus of all of these restaurants with a description of each dishes.
There will be a registration / login system to allow the user to grade and to put comments on the dishes / restaurants.
